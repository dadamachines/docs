# automat eurorack
{: .fs-9 }

Mechanical music inside your modular system — an **8 HP** eurorack module that brings the automat concept to patch cables, triggers and gates.

## Overview

Since 2017 the **automat toolkit** has let musicians, tinkerers and artists trigger solenoids, motors and LEDs with no coding or soldering. **automat eurorack** takes that same mechanical-music platform and drops it straight into a modular rig — instead of sending MIDI notes, eurorack users send **triggers or gates** to play physical objects: cans, boxes, metal sheets, percussion, anything that makes a sound.

---

## Key features

- **6 trigger / gate inputs** — patch any trigger or gate sequencer directly into the module.
- **6 universal DC outputs (12–24 V)** — drive solenoids, motors and other mechatronic elements.
- **TRS MIDI Type A input** — optional MIDI workflow alongside the CV triggers.
- **USB Type-A host port** on the front panel — plug in a Novation **Launchpad** for direct, computer-less performance.
- **Clock + reset input** — for synchronising the on-board sequencer with the rest of your rack.
- **8 HP** wide — compact enough for any skiff or case.

---

## Standalone mode

Switch the module into standalone mode and it runs on-board sequencing firmware — the same idea as the [automat controller MK2]({{ site.baseurl }}/automat-toolkit/automat-controller-mk2/) brings to the desktop. Plug in a Launchpad and the eurorack module becomes a fully standalone mechanical instrument — step-sequence the six outputs straight from the grid, no patch cables required.
