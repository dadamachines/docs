# Effects (FX)

The **tbd 16** has two global FX buses: **FX1** and **FX2**. Access them via the Mixer Select Overlay (hold `FUNC4`, press **Step 6**) or navigate from Track 16 with `→`.

## FX1

FX1 has **3 pages** of parameters (12 parameters total). The page names and parameter names are defined by the loaded FX preset.

![FX1 — page 1](../../../images/tbd-16/fx_delay.png){: .screenshot }

![FX1 — page 2](../../../images/tbd-16/fx_page1.png){: .screenshot }

- **Knobs 1–4** — adjust the 4 parameters on the current page
- **↑/↓** — switch between pages

Each parameter displays its short name and value. Parameters with automation show:
- `L` prefix for LFO modulation
- `R` prefix for random modulation

## FX2

Navigate from FX1 with `→`. FX2 has **1 page** of parameters (4 parameters).

![FX2 — page 1](../../../images/tbd-16/fx2_page0.png){: .screenshot }

## Navigation

| Action | Result |
|:-------|:-------|
| `←` at FX1 | Go to Track 16 Sound screen |
| `→` at FX1 | Go to FX2 |
| `←` at FX2 | Go to FX1 |
| `→` at FX2 | Go to [Master]({{ site.baseurl }}/docs/mixer-and-effects/master/) |
| `↑` / `↓` | Switch pages within FX1 |
{: .dada-minimal-table }

## FX Sends

{: .note }
The amount of signal sent to each FX bus is controlled per-track from the [Track Mixer]({{ site.baseurl }}/docs/sound-and-tracks/track-mixer/) or the group/track [Mixer]({{ site.baseurl }}/docs/mixer-and-effects/mixer/) screen. Global FX return levels are on the [Global Sound]({{ site.baseurl }}/docs/mixer-and-effects/global-sound/) screen (Delay and Reverb knobs).
