# Master Output

The Master screen controls the final stereo output processing. Access it via the Mixer Select Overlay (hold `FUNC4`, press **Step 5**) or navigate from FX2 with `→`.

![Master Output — parameters](../../../images/tbd-16/master_page0.png){: .screenshot }

## Parameters

The Master screen has multiple pages of parameters. The page names and parameter names are **defined by the loaded master preset** — they are not fixed.

- **Knobs 1–4** adjust the 4 parameters on the current page
- **↑/↓** switches between pages

![Master — page 2](../../../images/tbd-16/master_page1.png){: .screenshot }

Each parameter displays its short name and current value. Parameters with automation show `L` (LFO) or `R` (random) prefixes.

## Navigation

| Action | Result |
|:-------|:-------|
| `←` | Go to FX2 |
| `→` | Go to Track 1 Sound screen |
| `↑` / `↓` | Switch pages |
