# WebUI

The **tbd 16** includes a **built-in web interface** for sound design, sample management, and preset organization. Any device on the same network — laptop, phone, or tablet — can open it in a browser.
{: .fs-6 .fw-300 }

## Accessing the WebUI

1. Connect the **tbd 16** to WiFi (see [Settings → WiFi](../project-and-settings/wifi-settings)) or to your computer via USB.
2. Open a browser and navigate to the device address (shown on the WiFi settings screen).
3. The main WebUI loads with tabs for **Plugins**, **Data** (File & Sample Manager), **Presets** (Preset & Macro Manager), and a link to the **System Updater** in the footer.

---

## What's in the WebUI

| Tool | Purpose |
|:-----|:--------|
| [File / Sample Manager](file-manager) | Browse, upload, and organize audio samples and kit files. |
| [Preset / Macro Manager](preset-macro-manager) | Create and edit macros and manage performer presets. |
{: .dada-minimal-table }

For firmware updates via the WebUI, see the separate [Firmware Updates](../firmware-updates) page.

---

## Configuration overlay

Most screens expose a **configuration overlay** (cog icon) with quick access to:

- WiFi network settings
- System information
- Update channel (Stable / Staging)
- Link to the **System Updater** — see [Firmware Updates](../firmware-updates)
