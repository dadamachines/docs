# Utilities

System utilities and diagnostic tools.
{: .fs-6 .fw-300 }

**FUNC1 → Settings → Utilities**

![Utilities](../../../images/tbd-16/utilities.png){: .screenshot }

| Item | Description |
|:-----|:------------|
| **Button tester** | Interactive button and encoder input debugger — useful when troubleshooting unresponsive controls |
| **MIDI debugger** | Monitor incoming MIDI messages in real time |
| **Reboot P4** | Restart the audio engine only |
| **Reboot Pico (and P4)** | Full system restart |
