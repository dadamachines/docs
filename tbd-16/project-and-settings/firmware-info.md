# Firmware Info

View the installed firmware versions.
{: .fs-6 .fw-300 }

**FUNC1 → Settings → Firmware Info**

![Firmware Info](../../../images/tbd-16/firmware_info.png){: .screenshot }

| Field | Description |
|:------|:------------|
| **P4 FW** | Audio engine firmware version |
| **Pico FW** | Sequencer firmware version |
| **WebUI** | Web interface version |
| **Last** | Last update status: `None`, `Pico updated`, `P4 updated`, or `P4 + Pico updated` |
{: .dada-minimal-table }

To update your firmware, see [Firmware Updates](../firmware-updates).
