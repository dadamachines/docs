# WiFi Settings

Choose how the **tbd 16** connects to your computer or network.
{: .fs-6 .fw-300 }

**FUNC1 → Settings → WIFI Settings**

![WiFi Settings](../../../images/tbd-16/wifi_settings.png){: .screenshot }

The screen shows the current mode, SSID (if applicable), and mDNS name you can use to reach the WebUI in a browser.

| Mode | Description |
|:-----|:------------|
| **Access Point (open)** | The **tbd 16** creates its own open WiFi network you can join from any device |
| **USB Network (NCM)** | Connect over a USB-C cable — the computer treats the device as a network interface (IP: 192.168.4.1) |
{: .dada-minimal-table }

Select a mode and press **→** to switch. The device reboots to apply the change.

Once connected, open the [WebUI](../webui/) in a browser.
