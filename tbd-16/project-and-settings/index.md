# Project & Settings

Everything under the **FUNC1** menu on the device — project management (load, save, new) and all system configuration screens.
{: .fs-6 .fw-300 }

---

## How the menu is organised

Press **FUNC1** to open the Project menu. This section mirrors that menu one-to-one.

| Page | What it covers |
|:-----|:---------------|
| [Project](project) | Load, save, and manage projects |
| [MIDI Settings](midi-settings) | MIDI port routing, channel mapping, Ableton Link |
| [Display Settings](display-settings) | OLED, LED brightness, and system toggles |
| [WiFi Settings](wifi-settings) | Access Point or USB Network mode |
| [Utilities](utilities) | Button tester, MIDI debugger, reboot |
| [Firmware Info](firmware-info) | Installed firmware versions |
{: .dada-toc-table }
